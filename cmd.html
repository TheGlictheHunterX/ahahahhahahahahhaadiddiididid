Comprehensive Guide to Nmap, SQLmap, and Nikto Commands

Nmap Commands

Network Mapper (Nmap) is a powerful tool for network discovery and security auditing. Here are some basic and useful Nmap commands:

nmap -sP 192.168.1.0/24

Ping scan to find live hosts on a network.

nmap -p 80,443 192.168.1.1

Scan specific ports (80 and 443) on a target.

nmap -sV -p 1-100 192.168.1.1

Service version detection on ports 1-100.

nmap -O 192.168.1.1

Operating system detection.

nmap --script vuln 192.168.1.1

Run vulnerability scanning scripts against the target.

SQLmap Commands

SQLmap is an automated tool for detecting and exploiting SQL injection vulnerabilities. Here are some common SQLmap commands:

sqlmap -u "http://example.com/page.php?id=1" --batch

Basic SQL injection detection and exploitation.

sqlmap -u "http://example.com/page.php?id=1" --dbs

Enumerate databases on the target.

sqlmap -u "http://example.com/page.php?id=1" --tables -D database_name

List tables in a specific database.

sqlmap -u "http://example.com/page.php?id=1" --dump -D database_name -T table_name

Dump data from a specific table.

sqlmap -u "http://example.com/page.php?id=1" --os-shell

Get an OS shell if the database user has sufficient privileges.

Nikto Commands

Nikto is a web server scanner that tests for dangerous files, outdated server software, and other security issues. Here are some examples:

nikto -h http://example.com

Scan a web server for vulnerabilities.

nikto -h http://example.com -p 8080

Scan a web server on a specific port (e.g., 8080).

nikto -h http://example.com -Tuning 1

Run specific types of tests (e.g., tests for vulnerabilities).

nikto -h http://example.com -output result.txt

Save the scan results to a file (result.txt).

nikto -h http://example.com -Cgidirs

Check for potentially vulnerable CGI directories.
